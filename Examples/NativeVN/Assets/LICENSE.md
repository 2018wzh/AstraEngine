# NativeVN 技术验收素材许可

`Backgrounds/apartment-night.png` 与 `Characters/lin-yao.png` 是为 AstraEngine NativeVN 技术验收样例生成的原创素材。AstraEngine 项目及其下游可复制、修改和再分发这些文件。

`Bgm/`、`Movies/` 与 `Se/` 下的文件由确定性 FFmpeg 流程生成，按本文件同等条款授权。它们不包含商业游戏 payload。

`Voice/cc0-voice-sample.mp3` 与 `Engine/Fixtures/PublicDomainMedia/t-rex-roar.mp3` 内容相同，来源为 MDN interactive-examples，使用 CC0-1.0。

`Fonts/Poppins-Regular.ttf` 来自 google/fonts 的 Poppins family，使用 SIL Open Font License 1.1；完整许可见 `Fonts/OFL.txt`。

所有 sidecar 均记录 asset id、相对 source path、SHA-256、许可和目标 profile。构建过程不联网。
