# EngineCore Production Checklist

Status: Goal-aligned runtime / AstraVN / player audit ledger
Last audited: 2026-06-19

Purpose: track whether EngineCore can support normal player-facing VN gameplay from source content to packaged release across the full target platform set: Windows, Linux, macOS, iOS, Android, and Web. Current runtime evidence is useful but does not complete the final cross-platform gate until both NativeVN and TsuiNoSora have full-playthrough evidence on every target platform.

Legend:

- [x] Code-backed complete in the current source tree, with automated test, CLI, or committed report evidence.
- [~] Partial, foundation-only, narrower than the final production target, or design-contract-only.
- [ ] Planned or not implemented.

Rule: no item is complete from documentation alone. Every `[x]` row must name checked-in source and verification evidence. The row format is:

`- [status] ID - Requirement. Goal: <design goal>. Supports: <supporting elements>. Gate: <acceptance gate>. Design: <docs>. Code: <files/targets or none>. Evidence: <tests/CLI/reports or none>. Gap: <remaining production work>.`
